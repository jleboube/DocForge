            How To Import Content
            
          
            
            
            1. Folder Documents (PDF/TXT/MD/etc.)
            
            
            Go to Sources, add a source with type folder, and enter a container-visible path (example: /workspace/sample-data/docs).
            

            
            2. Kindle Highlights
            
            
            Add a source with type kindle and point to My Clippings.txt. Run scan/reindex from Admin or wait for workers.
            

            
            3. Webpages
            
            
            Use Web Capture to paste article text/HTML or use Import URL for server fetch ingestion.
            

            
            4. OneNote
            
            
            Use source type onenote as a placeholder. Current MVP tracks the source, while parser integration is scaffolded for expansion.
            

            
            5. Tagging Artifacts
            
            
            Set tags when creating highlights/clips, or click Set Tags on documents/highlights lists. Use Find Tag in Search.
